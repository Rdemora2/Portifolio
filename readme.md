# Sobre o projeto

Projeto de meu portifólio pessoal desenvolvido com tecnologias como: docker, django e react.

A aplicação consiste em exibir alguns de meus projetos de minha autoria e tem também alguns validadores feitos por mim, além de trazer as principais informações sobre minha experiência

# Como executar o projeto
Pré-requisitos: Docker

```bash
# clonar repositório
git clone https://github.com/Rdemora2/Portifolio

# buildar os containers docker
docker compose build

# subir os containers
docker compose up
```
